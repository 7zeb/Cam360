# Cam360
A Minecraft mod for capturing 360 degree images.
![GitHub Downloads (all assets, all releases)](https://img.shields.io/github/downloads/7zeb/Cam360/total)

## What is this?
Cam360 is a Fabric Mod that let's you take 360 images in Minecraft when you press a key, the mod will automatically tilt to all the different angles and take a screenshot.

## Requirements
To install Cam360 you need 
- Minecraft Java Edition version 1.21.11
- Java (use Java 21 or later)
- Fabric 0.18.0 or later (0.18.6 tested)

**This is a client side mod**
## How to install
(this is still being written, it will probably get updated soon)
If you do not have Fabric installed, download it from the [Official Website](https://fabricmc.net/)

**The Fabric API is also recomended, download it [Here](https://github.com/FabricMC/fabric/releases)**
After you do that, run the .jar file and follow the steps.

**Windows steps will be next**
- Open run (or press Win + R)
- Type "%appdata%"
- go into ".minecraft"
- go into "mods"
- drag and drop both the Fabric API and Cam360 into the folder
- Run Minecraft

## How to use
to use the mod, press F12 anywhere in game, after you do it will automatically rotate your player around and the screenshots will be saved in the "screenshots360" folder inside of ".minecraft"
